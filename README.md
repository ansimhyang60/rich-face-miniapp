# 🤑 리치 페이스 (Rich Face AI) 

> 얼굴 관상 분석(AI)과 실제 금융 데이터(Toss)를 결합하여 '부자의 상'을 진단하고 팩폭을 날리는 도파민 자극형 토스 미니앱입니다.

## 📱 핵심 비즈니스 로직 (BM)
- **현타 리포트 (결제 유도)**: 천만장자 관상 결과를 보여준 뒤, 토스 실제 소비 내역을 분석해 '소비 방어력 최하 등급'을 알리며 100원 결제를 유도 (Toss Payments 위젯 연동).
- **마이데이터 연동 보상**: 토스 브릿지를 통해 마이데이터 연동 동의 시, 무료 재검사 혜택(크레딧) 지급.
- **보상형 광고**: 광고 영상 시청 시 크레딧 추가 지급.

## 🛠️ 기술 스택 (Tech Stack)
- **Frontend**: React, Vite, TypeScript
- **State Management**: Zustand
- **UI/UX**: Vanilla CSS (토스 디자인 시스템 룩앤필 및 애니메이션 적용)
- **Deployment**: GitHub 연동 자동 배포 (Netlify -> 추후 Toss Developers)
